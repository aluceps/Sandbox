# Sandbox
MVPアーキテクチャを模倣したSandboxプロジェクトです。  
ConnpassのAPIからイベント情報を取得して表示するだけの機能を持ちます。

## Plugins
- OkHttp3 and Retrofit2 - Square
- Dagger2 and Gson - Google
- RxJava2 and RxAndroid2 - ReactiveX
- Timber - JakeWharton
